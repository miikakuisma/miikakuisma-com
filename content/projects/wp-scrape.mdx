Visits a website made with WordPress and attempts to find out which plugins and version numbers are used, and if the plugins are the latest versions available. This information is then used for generating recommended actions regarding security by using Chat GTP.

##### Stack

- NextJS
- React
- TypeScript
- TailWind CSS
- Open AI API
