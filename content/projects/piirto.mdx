Piirto app is super simple drawing app based on Tatami Canvas, the engine that runs Sumopaint. When magic button is pressed it will send your drawing to Open AI's Vision API for analysis, then from the analysis it will generate new image and loads it on the canvas.


##### Stack

- NextJS
- React
- TypeScript
- TailWind CSS
- Open AI API
- RadixUI
- Vercel Blob
